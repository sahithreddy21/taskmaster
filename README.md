# ⚡ TaskMaster

TaskMaster is a responsive, dark-themed Kanban board and task management application built with **React**, **Vite**, and **Inline Styles**. It includes an interactive AI Assist panel designed to generate task descriptions and acceptance criteria effortlessly.

🔗 **Live Preview:** [Replace with your taskmaster-lyart-one.vercel.app link]

---

## ✨ Features

* **Dual Views:** Seamlessly switch between a Kanban Board layout and a structured List View.
* **Smart Analytics:** Real-time stats tracking for total, in-progress, completed, and urgent tasks.
* **AI Assist Panel:** Integrated helper to generate task summaries automatically.
* **User Profiles:** Quick simulated profile switching upon entry.
* **Fully Responsive:** Beautifully adapted for varying viewport sizes.
